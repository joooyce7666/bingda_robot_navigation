# Bingda Robot Navigation Package

## Overview

This repository is a navigation package for the **Bingda robot platform**, including SLAM and basic motion planning modules.

It is based on the original official implementation:

https://gitee.com/bingda-robot/robot_navigation

---

## Modifications

This repository does not introduce new algorithms.  
It only adjusts parameters of the original navigation package to better fit the experimental settings used in academic research.

Main modifications include:
- SLAM parameter tuning  
- Costmap and planner parameter adjustment  
- RViz configuration for experiments  

---

## Original Source

All core algorithms are provided by Bingda Robotics.  
Please refer to the original repository for full system details.
